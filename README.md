# Appwrite Skills Repository

## English

This repository contains a focused set of Appwrite skills for practical fullstack development.

It is designed for teams and builders who want consistent implementation patterns across:
- Next.js fullstack
- React with Vite
- React Native
- Web JavaScript and TypeScript
- Security architecture
- Supabase to Appwrite migration

### Project Goal

The goal is to make Appwrite adoption faster, safer, and more consistent in real projects.

These skills help you:
- apply a unified Appwrite workflow from one entry skill
- use production-oriented permission and database security patterns
- reduce setup mistakes across frameworks and environments
- migrate from Supabase with clear limits and validation steps

### Skill Layout

- `SKILL.md`: unified entry and routing skill
- `core/`: shared Appwrite integration baseline
- `frameworks/`: framework-specific implementation skills
- `security/`: database and permission hardening skill
- `architecture/`: fullstack system design skill
- `migrations/`: Supabase migration workflow skill

### 프로젝트 목표

이 프로젝트의 목표는 Appwrite 도입을 더 빠르고, 안전하고, 일관되게 만드는 것입니다.

이 스킬들은 다음을 돕습니다.
- 하나의 통합 스킬에서 전체 워크플로우 시작
- 운영 환경 기준의 권한 및 DB 보안 설계 적용
- 프레임워크와 환경별 설정 실수 감소
- Supabase 마이그레이션 시 제한사항과 검증 절차 명확화

### 스킬 구성

- `SKILL.md`: 통합 진입점 및 라우팅 스킬
- `core/`: 공통 Appwrite 개발 기준
- `frameworks/`: 프레임워크별 구현 스킬
- `security/`: DB/권한 보안 강화 스킬
- `architecture/`: 풀스택 시스템 설계 스킬
- `migrations/`: Supabase 마이그레이션 스킬
